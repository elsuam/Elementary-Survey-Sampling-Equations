# Elementary-Survey-Sampling-Equations

Scheaffer, R. L., W. Mendenhall, R. L. Ott, and K. G. Gerow. 2011. Elementary Survey Sampling. Cengage Learning. https://books.google.com/books?id=qKYJzgEACAAJ.
